Key findings

  • After stripping out inflation, the market's price has grown only ~2.4%/year — the big headline numbers are mostly inflation plus reinvested dividends.
  • Valuation (CAPE) statistically predicts the next decade's real returns (R² ≈ 0.11, p ≈ 1e-40); today's CAPE is in the top 5% of all history, implying below-average returns ahead.
  • Deep drawdowns recur — the worst real loss reached −81% in the 1930s; the VIX spikes at every crisis.
  • Gold's correlation with stocks is ≈ 0, making it a genuine diversifier.
Recommendation: Set realistic return expectations, diversify with low-correlation assets, and automate discipline (dollar-cost-average + rebalance).
